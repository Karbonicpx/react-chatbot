  <li><strong>Fast and lightweight</strong> React implementation</li>
</ul>

---

<h2><em>How It Works</em></h2>

<p>
The user types a message in the input field and sends it.  
The chatbot checks the input against a predefined set of responses.  
If a match is found, the chatbot returns the corresponding answer.  
If not, it sends a fallback default message.
</p>
